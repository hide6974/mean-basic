# 手作りMEAN Stack


1.macでフォルダ作成　以下手順で実施
git clone \https://github.com/naga3/mean-basic

2.以下必要なモジュールインストール
npm install express
npm install body-parser
npm install mongodb

3.アクセス
http://localhost:3000/


MEAN Stack(MongoDB+Express+AngularJS+Node.js)を試してみようと考えた次第です。
